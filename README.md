# TIMSmon

TIMS(dtims.tibero.com) 기술지원요청 자동 모니터링 스크립트

신규 기술지원 요청이 등록되면 브라우저 알림과 화면 패널로 즉시 알려줍니다.

---

## 파일 구성

| 파일 | 설명 |
|---|---|
| `monitor.js` | 메인 모니터링 스크립트 (콘솔 실행용) |

---

## 사용 방법

### 방법 1: 콘솔 실행

1. TIMS 로그인 후 **기술지원요청** 메뉴로 이동
2. `F12` → Console 탭 열기
3. `monitor.js` 전체 코드 붙여넣기 → Enter

### 방법 2: 북마크릿 (권장)

아래 코드를 북마크 URL로 저장 후, TIMS 탭에서 클릭:

```
북마크릿 코드는 monitor.js 내용을 한 줄로 압축하여 사용
```

> 저장 방법: 크롬 북마크바 우클릭 → 페이지 추가 → URL에 위 코드 붙여넣기

---

## 동작 원리

```
실행
 │
 ├─ 초기 1회: 현재 요청 목록 전체를 기준점(Set)으로 저장
 │
 └─ 30초마다 반복:
      API 호출 (POST /findTechServiceRequestList.screen)
       │
       ├─ 세션 만료 감지 → 패널 빨간불 + 알림 팝업 → 재로그인 후 자동 재개
       │
       └─ 신규 요청 감지 (기준점에 없는 요청번호)
            └─ 패널에 NEW 표시 + 브라우저 알림 팝업
```

## 신규 요청 판단 기준

- 각 요청 행의 `onclick` 속성에서 요청번호(`TD-XXX-날짜-번호`) 추출
- `Set`에 없는 번호 = 신규 요청
- 한 번 알림 발송 후 Set에 추가 → 중복 알림 없음

---

## 패널 기능

| 기능 | 설명 |
|---|---|
| 상태 표시 | 초록불(감시중) / 빨간불(세션만료) |
| 마지막 체크 시간 | 가장 최근 API 호출 시간 |
| 전체 건수 | 현재 조회된 요청 총 건수 |
| 신규 카운트 | 감시 시작 후 신규 등록된 건수 |
| 로그 영역 | 체크 이력 (최근 80줄) |
| 드래그 | 헤더 드래그로 패널 위치 이동 |
| 최소화 | `_` 버튼으로 패널 접기/펼치기 |

---

## 주의사항

- 브라우저 탭이 열려 있는 동안만 동작합니다
- 새로고침 시 스크립트가 초기화됩니다 (북마크릿으로 재실행)
- 세션은 30초마다 API를 호출하므로 자동 갱신됩니다
- 브라우저 알림 권한 허용 필요
